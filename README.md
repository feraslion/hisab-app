# HisabApp

تطبيق محاسبة يعمل بدون إنترنت (Offline) مع إمكانية المزامنة السحابية.

## المميزات
- إدارة المنتجات والباركود.
- إدارة العملاء.
- إنشاء فواتير البيع والشراء.
- يعمل بالكامل بدون إنترنت باستخدام Dexie.js.
- مزامنة اختيارية مع Supabase.
- بناء تلقائي لنسخ Windows (.exe) و Android (.apk) عبر GitHub Actions.

## كيفية التشغيل محلياً
1. `npm install`
2. `npm run dev`

## البناء (Build)
يتم البناء تلقائياً عند رفع الكود إلى GitHub في تبويب **Actions**.
